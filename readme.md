# A Fine-grained Adaptive Framework for Implicit Hate Speech

This repository contains a Fine-grained Adaptive Framework for Implicit Hate Speech. It integrates Prompt Tuning (PT), Iterative Optimization (itera), and Agentic Context Engineering (ACE).

## 📁 Project Structure

* **`PT/`**: Core module for Prompt Tuning. Includes code for fine-grained data partitioning and the final detection task for Shallow Hate.
* **`itera/`**: Iterative tuning module that utilizes itera method for progressive performance enhancement.
* **`ACE/`**: Agentic Context Engineering. Contains optimized prompts generated by intelligent agents for LLM inference.

---



## 🚀 Execution Guide

### Phase 1: PT Module
The code in the `PT` directory needs to be executed twice to complete the pipeline:

#### 🛠️ Environment & Installation

(1)Install all required Python packages using pip:
```bash
pip install -r requirement.txt
```
(2)This framework is built on OpenPrompt. Please follow these steps to configure the environment:
1. Clone and install the OpenPrompt library.
2. Overwrite the following files in your OpenPrompt installation with the provided files from this repository:
   * Copy `text_classification_dataset.py` to `Openprompt/openprompt/data_utils/text_classification_dataset.py`
   * Copy `pipeline_base.py` to `Openprompt/openprompt/pipeline_base.py`

(3)Download the following pre-trained models from Hugging Face and place them in the `models/` folder:
* **BERT-Chinese**: [bert-base-chinese](https://huggingface.co/bert-base-chinese)
* **RoBERTa-Base**: [roberta-base](https://huggingface.co/FacebookAI/roberta-base)

(4)Place your custom datasets in: `datasets/TextClassification/`

#### ▶️Run
1. **First Run**: Performs the initial fine-grained classification.
2. **Second Run**: Performs the final **Shallow Hate** category detection task.

**Run command:**
```bash
cd PT
python auto_run.py
```

### Phase 2: itera Module
This module builds upon the PT stage with iterative refinements.

**Prerequisite**: Before running, replace the soft tuning logic file:
```bash
cp sotf_tuning.py Openprompt/openprompt/soft_tuning.py
```
**Execution**: Follow the numerical order of the scripts in the `itera` folder to ensure the logic flows correctly.

### Phase 3: ACE Module
This module provides optimized instruction prompts for LLMs.

* **Usage**: Open the `final_prompts` file to retrieve the generated prompts.
* **Integration**: Combine these prompts and the data to be detected with your standard LLM API calling code for immediate inference.
